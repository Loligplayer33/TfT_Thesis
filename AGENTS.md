# Workspace Agent Entry Point

This workspace contains the thesis vault in:

- `TfT_Notes_Vault/`

If you are an AI agent working anywhere in this workspace:

1. Read `TfT_Notes_Vault/AGENTS.md` first.
2. Then read `TfT_Notes_Vault/CLAUDE.md` for fuller project context.
3. If the task touches Zotero imports, note templates, workflow docs, or shared AI documentation, also read:
   - `TfT_Notes_Vault/ai/zotero-import-template-guide.md`

Important:

- General agent conventions are defined in `TfT_Notes_Vault/AGENTS.md`.
- Shared workflow/context changes are not done until the relevant AI docs are updated.
- When shared AI docs or workflow/config files change, run:
  - `python3 scripts/validate_ai_docs.py`
